# Targetology Landing

Вузький персональний сайт послуг із Meta Ads та Google Ads для Аліни Силки.

## Структура

- головний екран із фото й двома CTA;
- системний підхід до реклами;
- слайдер форматів роботи;
- оффер спікерства;
- кваліфікація бюджету від $600/міс.;
- об’ємна карусель кейсів;
- соціальний доказ, критерії співпраці та FAQ;
- квіз-форма й окрема Thank-you page.

## Доступ

Сторінка опублікована на Cloudflare Pages і відкрита для індексації:

- `index,follow` у метаданих;
- `Allow: /` у `robots.txt`;
- `sitemap.xml` для пошукових систем.

Посилання: https://alina-silka.pages.dev/
